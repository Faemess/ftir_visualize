# FTIR Spectra Viewer
## پیش‌نیازها

* Python 3.10 یا جدیدتر
* سیستم‌عامل Windows، Linux یا macOS

---

## نصب

ابتدا مخزن پروژه را دانلود یا فایل‌های پروژه را در یک پوشه قرار دهید.

### 1. نصب وابستگی‌ها

در پوشه پروژه cmd را باز کرده و دستور زیر را اجرا کنید:

```bash
pip install -r requirements.txt
```

## اجرای برنامه

پس از نصب وابستگی‌ها:

```bash
python ftir_viewer.py
```
